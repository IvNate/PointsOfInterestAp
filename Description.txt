Описание Api
1. Метод POST /point - добавление новой точки интереса
Формат запроса:
{
	'name' : 'Пермский планетарий', 		//[string, обязательное поле]
	'description' : 'Описание планетария', 	//[string, обязательное поле]
	'latitude' : 58.019068, 				//[float, обязательное поле]
	'longitude' : 56.271742,				//[float, обязательное поле]
	'type' : 'Место'						//[string, обязательное поле]
}
Возвращаемое значение: JSON
Формат:
{	
	"id":1,
	"name":"Пермский планетарий",
	"description":"Описание планетария",
	"latitude":58.019068,
	"longitude":56.271742,
	"type":"Место"
}
Авторизация : доступен неавторизованному пользователю
Коды ошибок:
400 - Not valid request parameters.
404 - Point of interest type not found.

2. Метод PUT /point/{id} - обновление полей точки интереса
Формат запроса:
{
	'name' : 'Пермский планетарий', 		//[string, обязательное поле]
	'description' : 'Описание планетария', 	//[string, обязательное поле]
	'latitude' : 58.019068, 				//[float, обязательное поле]
	'longitude' : 56.271742,				//[float, обязательное поле]
	'type' : 'Место'						//[string, обязательное поле]
}
Возвращаемое значение: JSON
Формат:
{	
	"id":1,
	"name":"Пермский планетарий",
	"description":"Описание планетария",
	"latitude":58.019068,
	"longitude":56.271742,
	"type":"Место"
}
Авторизация : доступен неавторизованному пользователю
Коды ошибок:
400 - Not valid request parameters.
404 - Point of interest type not found.
404 - Point of interest not found.

3. Метод GET /points/all - запрос всех точек интереса в городе пользователя
Формат запроса:
{
	'limit' : 1, 					//[int, необязательное поле, по умолчанию 50]
	'offset' : 0, 					//[int, необязательное поле, по умолчанию 0]
	'city' : 58.019068, 			//[string, необязательное поле]}
Возвращаемое значение: JSON
Формат:
[{
	"id":1,
	"name":"Пермяк солёные уши", "description":"Описание",
	"latitude":58.013779,
	"longitude":56.245398,
	"type":"Памятник"
}]
Авторизация : доступен неавторизованному пользователю
Коды ошибок:
400 - Not valid request parameters.
404 - City not found.

4. Метод GET /points - запрос всех ближайших точек интереса
Формат запроса:
{
	'ip' : 127.0.0.1, 				//[string, необязательное поле]
	'radius' : 11, 					//[float, необязательное поле, по умолчанию 1 (км)]
Возвращаемое значение: JSON
Формат:
[{
	"id":1,
	"name":"Пермяк солёные уши", "description":"Описание",
	"latitude":58.013779,
	"longitude":56.245398,
	"type":"Памятник"
}]
Авторизация : доступен неавторизованному пользователю
Коды ошибок:
400 - Not valid request parameters.
404 - Ip not found.
